# 🏠 House Price Prediction using Machine Learning

## 📌 About the Project
This project is my first hands-on implementation of a Machine Learning model where I explored how different factors like number of rooms, locality indicators, and economic features affect house prices.

The goal was to understand the complete ML workflow — from data preprocessing to model building and evaluation.

---

## ⚙️ Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib  

---

## 🧠 Model Used
I used **Linear Regression** to predict house prices based on selected features.

---

## 📊 Results
- Achieved an R² score of around **0.66–0.73**
- Built a visualization comparing **actual vs predicted prices**

---

## 📈 Key Learnings
- Understanding real-world datasets  
- Feature selection and data cleaning  
- Training and evaluating ML models  
- Importance of visualization in analysis  

---

## 🚀 Future Improvements
- Try advanced models (Random Forest, XGBoost)  
- Improve accuracy with better feature engineering
